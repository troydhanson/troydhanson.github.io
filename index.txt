.Projects
This is a listing of my open source projects. Everything is in C for Linux.
Well, almost everything.

:i: http://troydhanson.github.io/
:t: https://github.com/troydhanson/

* {i}uthash[uthash] a hash table for C structures, with linked lists and dynamic strings in C
* {t}network[network] examples of TCP, UDP, DNS, raw sockets, CIDR math, etc, in C
* {i}pmtr[pmtr] a light process supervisor with everything in one config file 
* {t}fluxcap[fluxcap] a tool to replicate, aggregate and retransmit network taps
* {i}tpl[tpl] binary serialization for C

These are rougher around the edges:

* {t}kernel[kernel] module examples
* {t}sound[sound] basic PCM capture, playback, synthesis
* {t}info-theory[data] compression, error correcting codes, Bloom filters
* {t}boiler[misc] heap of examples e.g. epoll, mmap, serial, scripts
